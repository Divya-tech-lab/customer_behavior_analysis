# customer_behavior_analysis
Analyzed 3,900 customer transactions to uncover shopping patterns and business insights. Performed data cleaning and feature engineering in Python, conducted SQL analysis on revenue, customer segments, and product trends, and built a Power BI dashboard to visualize insights and support data-driven decisions.
📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from data loading to delivering actionable insights. It includes data cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and reporting.
